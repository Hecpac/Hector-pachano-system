# Design Library v1

Carpeta base para prompts, estilos y ejemplos reutilizables.

## Estructura
- `prompts/`: prompts por tipo de pieza.
- `styles/`: tokens/guías rápidas de estilo.
- `examples/`: capturas o enlaces aprobados.

## Convención de nombres
- `YYYY-MM-DD_tipo_objetivo.md`
- Ejemplo: `2026-02-26_hero_lead-b2b.md`
