# Reverse prompts (listos para usar)

## 1) Prioridad diaria (5 min)
"Con base en mis metas (PHD primero, QTS segundo), dime la accion #1 de mayor impacto para hoy. Dame solo: objetivo, primer paso (<5 min), riesgo principal y criterio de terminado."

## 2) Cuando hay bloqueo
"Estoy bloqueado en [tema]. Propon 3 rutas (A/B/C) con trade-off costo/tiempo/riesgo y recomienda una con plan de ejecucion hoy."

## 3) Para automatizar trabajo repetitivo
"Observa esta tarea repetitiva: [tarea]. Propon una automatizacion minima viable (MVP), como validarla hoy y rollback si falla."

## 4) Para mejorar calidad
"Revisa este resultado: [resultado]. Dime 3 mejoras de mayor impacto y aplica primero la de mejor relacion impacto/esfuerzo."

## 5) Para seguridad antes de ejecutar
"Antes de ejecutar, dame plan paso a paso, riesgos, permisos que tocaria y confirmacion explicita de que no expondras credenciales ni datos sensibles."

## 6) Para cerrar el dia
"Haz cierre operativo: que se avanzo, que queda bloqueado, siguiente accion unica para manana y riesgo que debo vigilar."
