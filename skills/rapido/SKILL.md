# Rápido

Aplica un formato de ejecución breve.

## Instrucciones

1. Responde en formato: **qué haré → evidencia mínima → siguiente paso**.
2. Mantén la salida corta (máx. 6 bullets o 8 líneas).
3. Prioriza acciones verificables sobre explicación teórica.
4. Si faltan datos críticos, haz máximo 2 preguntas puntuales.
5. Si el usuario incluyó texto tras el comando, trátalo como la tarea a ejecutar en este modo.
