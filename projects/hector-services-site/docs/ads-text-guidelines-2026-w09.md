# Ads Text Guidelines — Semana W09 2026

## Objetivo
Alinear copy de anuncios con intención de campaña para subir calidad de lead y reducir fricción entre Ads → landing → ventas.

## 1) TOFU (Awareness)
- **Promesas permitidas:** diagnóstico, claridad, visibilidad de cuellos de botella.
- **Evitar:** “garantizado”, “instantáneo”, “sin esfuerzo”.
- **Tono:** educativo + autoridad práctica.

**Plantilla:**
Problema visible + señal de oportunidad + CTA de bajo compromiso.

## 2) MOFU (Consideración)
- **Promesas permitidas:** método, roadmap, fases, casos.
- **Evitar:** superlativos absolutos sin evidencia.
- **Tono:** consultivo + técnico-comercial.

**Plantilla:**
Problema + método en 3 pasos + prueba breve + CTA a diagnóstico.

## 3) BOFU (Decisión)
- **Promesas permitidas:** entregables concretos, próximos pasos, tiempos de arranque.
- **Evitar:** urgencia artificial o descuentos sin contexto.
- **Tono:** directo + confiable.

**Plantilla:**
Resultado esperado + alcance + CTA único de cierre.

## Claims que requieren evidencia
- “+X% conversión”
- “-X% tiempo operativo”
- “X semanas para implementar”

Siempre incluir caso/fuente interna trazable.

## Checklist antes de publicar
1. ¿Existe una sola promesa principal?
2. ¿El H1 de landing coincide con el anuncio?
3. ¿Hay prueba visible (caso, proceso, benchmark)?
4. ¿CTA único y claro?
5. ¿El tono coincide con TOFU/MOFU/BOFU?

## Mensajes base recomendados
- **TOFU:** “Detecta 3 fricciones que frenan tus leads B2B en menos de 15 min.”
- **MOFU:** “Sistema web + automatización con KPI semanal y validación real.”
- **BOFU:** “Agenda diagnóstico y sal con plan P0/P1 de 30 días.”
